A Quick Guide to Cyber Idols

Thomas Wu and Charles Wang Lv1
  2024-05-10 11:00 2024-05-10 11:00 Created   2024-05-10 11:06:44 2024-05-10 11:06:44 Updated  

Since the last blog I have received many requests on deploying stable diffusion, and the majority of them are anonymous. Since you guys are so willing to devote yourselves to art, here is a detailed guide on how to deploy a localized stable diffusion model on your device. And you can apply these techniques not only undress images but generate your own Cyber Idols.

Stable Diffusion WebUI

I have been using Stable Diffusion WebUI to generate AI pictures. It is a very popular, open source tool used for AI graphics. Given a prompt, the model would generate the “desired” picture. Note that this model heavily relies on GPU performance and memory.

Deployment Guide

Basic Environment:

  1. Nvidia/AMD GPU driver UP TO DATE
  2. Python 3.10 (newer version might not working well, I highly recommend you degrade your python if it’s 3.11 or 3.12)
  3. Git environment

Note that python and git environments do not come preinstalled. If you don’t have such an environment and are not willing to /fail to install one and still insist to proceed with the installment(there are numerous guide on google), you do not belong to the audience group of this blog. I don’t write blogs for lazy ass fools.

Stable Diffusion Model:
To generate pictures with Stable Diffusion there is to mandatory basic model:

  1. Face adjustment model GFPGAN
  2. Models related to drawing (depends on your choice)

GFPGAN can be found on github, as always:
https://github.com/TencentARC/GFPGAN

There are various models related to drawing, and can be found on multiple websites, such as
https://huggingface.co/models
https://civitai.com

Here are some recommended models:

  1. Stable Diffusion: For realistic people and animals
  2. Chilloutmix: Realistic model mixed with anime style
  3. Anything: For anime/comic style
  4. Wifu Diffusion: Comic style.
    You may filter on Civitai and download any model you like. Make sure that the model file type is in ckpt or safetensors

Just select the “Checkpoint” in model type here will be good

Install Stable Diffusion:

  1. Install the code from github, using the code:
  1. Move GFPGANv1.4.pth downloaded ealier into the designated folder(the stable diffusion folder)

  1. Download the relating dependent libraries, type bat for windows and sh for linux.

Every time it is run, the latest versions of various libraries will be synchronized through git. When it is run for the first time, various dependent libraries will be downloaded. It is a bit large and involves github. If it is fast, it will take about 30 minutes. If it is slow, it may be very slow. You may fail this step because of wrong python version(real experience of author)

If the web link is displayed, it means that all dependencies have been downloaded successfully. It should be noted that you may encounter many errors when relying on torch, so you may consider downloading and installing it manually.

  1. Installing the drawing model: Place the model in the following directory:
    /models/Stable-diffusion/

  1. If you can open the link (commonly 127.0.0.1:7860) and can use it properly, you successfully finish setting up the tool on your computer!

Simple User Guide

Prompt

When using Stable diffusion, the most key factor that influences the results are prompts. There are postive prompt and negative prompts.

Regardless of model they spawn pictures with prompts. There are several key features:

  1. Must be English input. You may use word, such as “girl, long hair”, or a sentence such as “a girl with long hair”, or use names of artist to generate in their style, or even use a anime character’s name.
  2. You may adjust a prompt’s weighting by bracketing them. The more you bracket, higher the weighting.

Below we also share a general negative prompt to avoid unwanted output, such as broken limbs.(Shared by Hentai Diffusion)

Generating Image From Image

As mentioned earlier, stable diffusion generates pictures through prompt. In addition to text, it also provides a picture generation function that provides picture generation. You can generate images by uploading images + keyword combinations.

Using Shared Prompt and Parameter Setting

After understanding the previous things, you should be able to generate some various types of pictures, but to be honest, many of the generated pictures are very poor. At this time, you can go to civitai to browse and find some that others have shared.

Just choose one category you like, click on it and scroll down. You can see many pictures generated. Choose a image you like, click on the ! in lower right corner, you can see the parameters and prompt used for this image

Conclusion

Having fun generating your own idol!
