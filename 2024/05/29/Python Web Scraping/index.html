Web Scraping with Python

Introduction

Imagine you need to know about information on the internet, such as prices of a product, many pictures of a specific person or etc. Obviously you will not collect the info yourself as it is extremely labor intensive, and those data are massive (i’m talking about texts and pictures in Gigabytes). Thus you write a program, and let the program collect data according to your instructions. This is scraping, and the search engines we know such as google and bing are also fundamentally scraping.

Is it legal? Many may ask. As scrapings are only just programs, many of them are open source, there is no risk of using them. However, the misuse of such technology will lead to felonies or lawsuits. Examples of misuse include collecting sensitive information on government/bank/organization sites and DDOS attacks (massively using scraping). Usually, as long as the scraping doesn’t influence a website’s daily operation, nor for commercial purposes, the admin of that website will probably just ban your IP address or your account.

Most websites have a “robots” protocol, located in the root directory of that website. The protocol tells you things you can collect and things you cannot. An example:

https://www.google.com/robots.txt

The robots protocol is more like a friendly reminder/agreement/rule among the website and the user. It is not a legal significant documentation, so following it or not really depends on you.

Why Python for Web Scraping?

When people think of web scraping, Python often comes to mind. But why is Python the preferred choice for many? Here are a few reasons:

  1. Development Efficiency: Python allows concise code, and you can accomplish complex scraping tasks with just a few lines.
  2. Performance Requirements: Web scraping is often I/O-bound (waiting for network requests), so Python’s execution efficiency is sufficient.
  3. Rich Third-Party Libraries: Python has excellent third-party libraries like requests, beautifulsoup, and selenium.

Environment Setup

To get started with web scraping in Python, follow these steps:

  1. Install Python: Download the appropriate Python installer from the official website and install it.
  2. Install pip: Pip is Python’s package manager, and it usually comes bundled with Python installations.
  3. Install Required Libraries: Install the requests and beautifulsoup4 libraries using the following commands: pip install requests, pip install beautifulsoup4

Using Requests for Web Scraping

The requests library is widely used for web scraping. It handles network requests and responses. Here’s a basic example of using requests:

  1. Import the requests module
    import requests

  2. Make a GET request to Baidu
    response = requests.get('https://www.baidu.com/')

  3. We now get a response object named response, from which we can get a lot of information

Of course, most websites will require you to identify yourself. When we normally visit a website, there will be a request header (headers) information, which contains your browser, encoding and other content. The website will use this information to determine your identity, so we usually add a header when writing crawlers;

Beautifulsoup

After we get the HTML code of the entire page through requests, we have to process it further, because we often only need a small part of the data on the entire page, so we need to parse the HTML code of the page and filter out the data we want, and this is when beautifulsoup comes in handy. Beautifulsoup then locates it by tag + attribute. For example, if we want the Baidu logo, we check the HTML code of the page and we can find that the logo image is under a div tag and the attribute class=index-logo-srcnew.
So if we need to locate the logo image, we can use div and class=index-logo-srcnew to locate it.

    1. F12 opens the console and selects the element tag to view the page code;
    2. Click the arrow in the upper left corner of the console, and then click the information we need on the page. We can see that the page code in the console jumps directly to the corresponding position;
    3. In the page code, select the tag upwards until all the information we need is included;
    4. Remember the tags and familiar information at this time, which will be used for later parsing and filtering data.
